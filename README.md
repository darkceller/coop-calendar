## 📖 專案概述 (Project Overview)
本專案為解決主婦聯盟合作社新竹分社每月「活動資訊散亂、海報製作耗時、資訊更新不及時」等行政痛點，專為分社量身打造的**「全自動化活動海報生成系統」**。

透過整合 Google Sheets API 與前端動態網頁技術，實現從「各站所填報」到「A3 印刷級海報產出」的無縫接軌。本系統大幅降低行政人員的手動排版工時，並確保跨站所資訊的即時性與精確度。

---

## ✨ 核心特色 (Core Features)
1. **雲端即時同步**：資料庫綁定 Google 試算表，免去反覆寄送校對稿的行政往返。
2. **智慧排版引擎**：支援網頁瀏覽 (RWD) 與 A3 紙本高畫質列印 (300dpi 級別) 雙模式。
3. **動態狀態判定**：自動判定活動日期，已過期活動自動加上「已結束」浮水印。
4. **地區分流產出**：支援按地區（桃園區/新竹區）自動過濾並產出專屬海報，實現資訊精準投放。
5. **零伺服器成本**：採用 Serverless 架構 (GitHub Pages + Google API)，維護成本極低。

## 🛠️ 技術架構 (Tech Stack)
* **前端介面**: HTML5, CSS3 (CSS Grid / Flexbox), Vanilla JavaScript (ES6+)
* **資料庫與後台**: Google Workspace (Google Sheets & Google Forms)
* **API 串接**: Google Visualization API (GViz)
* **海報渲染引擎**: `html2canvas.js` (DOM 轉高畫質圖片直出)
* **版本控制與代管**: Git, GitHub Pages
